# 🖼️ 切り取りくん

ブラウザ上で動作する画像トリミングツールです。サーバーにアップロードすることなく、完全にクライアントサイドで画像処理を行います。

## ✨ 特徴

- **完全クライアントサイド処理** - 画像はブラウザ上でのみ処理され、サーバーに送信されません
- **直感的なUI** - ドラッグ&ドロップで簡単に画像をアップロード
- **柔軟なトリミング** - 縦横比を指定してトリミング可能
- **リアルタイムプレビュー** - トリミング範囲を視覚的に確認しながら調整
- **画像操作** - ドラッグで位置調整、スライダーでサイズ調整
- **高品質出力** - 指定したサイズで高品質な画像を出力

## 🎯 機能一覧

### 縦横比選択
- 自由（任意の比率）
- 1:1（正方形）
- 4:3（横長）
- 16:9（ワイド）
- 3:4（縦長）
- 9:16（縦長ワイド）

### トリミングサイズ指定
- 幅と高さをピクセル単位で指定可能
- 空欄の場合はトリミングエリアのサイズで出力

### 画像操作
- **ドラッグ&ドロップ** - 画像の位置を自由に調整
- **サイズ調整スライダー** - 10%〜300%の範囲で画像サイズを調整
- **トリミングエリア** - 半透過オーバーレイで範囲外を視覚化
- **リサイズハンドル** - トリミングエリアの角や辺をドラッグしてサイズ変更可能
  - 縦横比が選択されている場合は、その比率を維持したままリサイズ
  - 「自由」選択時は、任意のサイズに変更可能

### ボタン機能
- **トリミング** - 指定した範囲で画像を切り取り
- **やり直し** - トリミング前の状態に戻す
- **ダウンロード** - トリミングした画像をダウンロード
- **クリア** - アップロードした画像を削除して最初からやり直し

## 🚀 使い方

1. **画像のアップロード**
   - 画像をドラッグ&ドロップ、または「ファイルを選択」ボタンをクリック
   - 対応形式: JPEG, PNG, GIF, WebP など

2. **トリミング設定**
   - 縦横比を選択（デフォルトは「自由」）
   - 必要に応じてトリミングサイズを指定

3. **画像の調整**
   - 画像をドラッグして位置を調整
   - スライダーで画像サイズを調整
   - トリミングエリア（白枠）の角や辺をドラッグしてサイズ変更
   - トリミングエリア内に切り取りたい部分を配置

4. **トリミング実行**
   - 「トリミング」ボタンをクリック
   - プレビューで確認し、必要に応じて「やり直し」で調整

5. **ダウンロード**
   - 「ダウンロード」ボタンで画像を保存
   - ファイル名: `kiritori-[タイムスタンプ].png`

## 💻 技術仕様

- **HTML5 Canvas API** - 画像の描画とトリミング処理
- **FileReader API** - ローカルファイルの読み込み
- **Drag and Drop API** - ドラッグ&ドロップ操作
- **純粋なJavaScript** - フレームワーク不要
- **レスポンシブデザイン** - スマートフォン・タブレット対応
